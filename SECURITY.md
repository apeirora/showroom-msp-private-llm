# Security Policy

## About This Project

This repository contains a **demo operator** built for the [Apeiro](https://apeirora.eu) architecture. It is intended solely to showcase Platform Mesh capabilities and is **not designed for production use**.

## Reporting Vulnerabilities

If you discover a security issue, you may open a GitHub issue or contact the maintainers directly. However, please be aware of the following:

## No Guarantee of Security Fixes

This project is provided on a best-effort basis. **No security fixes are guaranteed.** Because this is a demonstration operator, vulnerabilities may not be triaged, patched, or released on any particular timeline — or at all.

Use this software at your own risk and do not deploy it in environments where security is a concern.
